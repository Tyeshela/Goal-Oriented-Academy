Goal-Oriented Academy

ჩვენ ვთვლით, რომ მომავალი დაფუძნებულია ონლაინ განათლებაზე. სტუდენტებს, მომავალში, ბევრი რამის სწავლა ინტერნეტ-რესურსებიდან მოუწევთ, ამიტომ ვთვლით, რომ ამ ასაკიდან ონლაინ დისციპლინის გამომუშავება, მათი წარმატების საწინდარია.

ლექციები ტარდება ონლაინ, ბავშვებისთვის ნაცნობ და საყვარელ პლატფორმაზე, Discord-ზე, მაღალხარისხიან, დაბუსთულ სერვერზე.

გაეცანით კურსის დეტალებს

რას გაეცნობით Goa-ში

  • Python

    Python არის მაღალი დონის, ზოგადი დანიშნულების პროგრამირების ენა.

  • HTML

    HTML არის სტანდარტული მარკირების ენა რომელიც შექმნილია ვებ ბრაუზერში გამოსატანად.

  • CSS

    CSS არის სტილის ენა, რომელიც გამოიყენება მარკირების ენის გასტილვისთვის (მაგ: HTML)

  • AI

    AI არის ინტელექტი რომელიც ადამიანზე მეტად ჭკვიანია და ის ადამიანის მიერაა შექმნილი.

  • Javascript

    JS არის პროგრამირების ენა, რომელიც არის მსოფლიო ქსელის ერთ-ერთი ძირითადი ენა

  • React

    React არის უფასო და Open-source front-end JavaScript ბიბლიოთეკა
